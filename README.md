이 어플리케이션을 사용하기 위해선

1. dev.js file을 config 폴더 안에 생성해주세요.
2. mongoDB 정보를 dev.js file안에다가 넣어주세요.
3. " npm install "을 root directory에서 입력해주세요. (백엔드 종속성 다운받기)
4. " npm install "을 client directory에서 입력해주세요. (프론트엔드 종속성 다운받기)
